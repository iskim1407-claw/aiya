# AiYa 프로젝트 개발 규칙

## 🚫 절대 금지
- **터치/버튼 방식 금지** - 아이가 터치할 수 없음
- **웨이크 워드 호출 방식만 사용** - "아이야~" 음성으로만 시작

## ✅ 필수 방식
- 자동 음성 감지 (웨이크 워드)
- 세션 기반 대화 (웨이크 워드 후 연속 대화)
- 30초 타임아웃으로 세션 종료

## 기술 스택
- Web Speech API: 웨이크 워드 감지용 (불완전해도 OK)
- OpenAI Whisper: 실제 대화 인식용 (정확함)
- OpenAI GPT-4o-mini: 대화 응답
- OpenAI TTS (nova): 음성 응답

## 대상
- 2-4세 아이
- 발음 불명확, 옹알이, 갑작스러운 주제 변경 대응 필요
